Видалення записів з використанням пов’язаних таблиць

При видаленні записів з таблиці можна використовувати інформацію з інших пов’язаних таблиць. У цьому випадку синтаксис запиту має вигляд: 

DELETE FROM таблиця_1
USING 
    таблиця_1 
    INNER JOIN таблиця_2 ON ...
WHERE ...;

Приклад

Видалити усіх авторів з таблиці author, у яких є книги з кількістю примірників менше 3. З таблиці book видалити всі книги цих авторів.

Запит:

DELETE FROM author
USING 
    author 
    INNER JOIN book ON author.author_id = book.author_id
WHERE book.amount < 3;

SELECT * FROM author;
SELECT * FROM book;
