Оператор CROSS JOIN

Оператор перехресного з’єднання або декартового добутку CROSS JOIN поєднує дві таблиці. У запиті замість ключового слова CROSS JOIN можна просто вказати обидві таблиці через кому. Порядок таблиць не має значення, оскільки цей оператор є симетричним. 

SELECT...
FROM
  таблиця_1 CROSS JOIN таблиця_2
...

або

SELECT
  ...
FROM
  таблиця_1, таблиця_2
...

Результат запиту формується так: кожен рядок однієї таблиці поєднується з кожним рядком іншої таблиці, утворюючи всі можливі комбінації рядків.

Приклад, запит: 

SELECT name_author, name_genre
FROM 
  author, genre;

Пояснення

1. Щоб згенерувати випадкову дату, можна до першого числа року ('2020-01-01') додати ціле випадкове число в інтервалі від 0 до 365.

Генерація випадкових чисел в інтервалі від 0 до 1 (не включно) здійснюється за допомогою функції RAND(). Якщо цю функцію помножити на 365, вона генеруватиме дійсні числа від 0 до 365 (не включно). Залишається лише відкинути дробову частину. Це можна зробити за допомогою функції FLOOR(), яка повертає найбільше ціле число, менше або рівне зазначеному числовому значенню. Таким чином, випадкове число від 0 до 365 можна отримати за допомогою виразу: 

FLOOR(RAND() * 365)

Важливо! Дати мають бути за 2020 рік, перше число року — 1 січня 2020 року.

2. Для додавання числа до дати використовується функція: 

DATE_ADD(дата, INTERVAL число одиниця_виміру)

де
одиниця_виміру (використовувати прописні літери) – це день (DAY), місяць (MONTH), тиждень (WEEK) тощо,
число – ціле число,
дата – значення дати або дати й часу.
Функція додає до дати зазначене число, виражене в днях, місяцях тощо, відповідно до заданого інтервалу, та повертає нову дату.

Наприклад:

DATE_ADD('2020-02-02', INTERVAL 45 DAY) повертає 18 березня 2020 року
DATE_ADD('2020-02-02', INTERVAL 6 MONTH) повертає 2 серпня 2020 року
