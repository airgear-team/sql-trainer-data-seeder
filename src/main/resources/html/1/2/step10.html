Вибірка даних з сортуванням

При вибірці можна вказувати стовпець або кілька стовпців, за якими потрібно відсортувати рядки. Для цього використовуються ключові слова ORDER BY. Рядки сортуються за першим стовпцем, а другий стовпець використовується для сортування рядків із однаковими значеннями першого. За замовчуванням сортування відбувається за зростанням (ASC), для спадання використовується DESC.

Стовпці після ORDER BY можна задавати:

  • назвою стовпця;
  • номером стовпця;
  • іменем стовпця (заданим після AS).

Приклад

Вивести назву, автора та ціну книг. Інформацію відсортувати за назвами книг в алфавітному порядку.

Запит:

SELECT title, author, price
FROM book
ORDER BY title;

Результат:

+----------------------+--------------------------+--------+
| title                | author                   | price  |
+----------------------+--------------------------+--------+
| Intermezzo           | Коцюбинський М.М.        | 540.50 |
| Кайдашева сімʼя      | Нечуй-Левицький І.С.     | 460.00 |
| Хмари                | Нечуй-Левицький І.С.     | 799.01 |
| Тіні забутих предків | Коцюбинський М.М.        | 670.99 |
| Земля                | Кобилянська О.Ю.         | 650.00 |
+----------------------+--------------------------+--------+

Аналогічний результат буде при використанні:

SELECT title, author, price
FROM book
ORDER BY 1;

Приклад

Вивести автора, назву та кількість книг, відсортованих за автором (алфавітно) та за спаданням кількості, для книг з ціною менше 750 грн.

Запит:

SELECT author, title, amount AS Кількість
FROM book
WHERE price < 750
ORDER BY author, amount DESC;

Результат:

+--------------------------+----------------------+------------+
| author                   | title                | Кількість  |
+--------------------------+----------------------+------------+
| Кобилянська О.Ю.         | Земля                | 15         |
| Коцюбинський М.М.        | Тіні забутих предків | 3          |
| Коцюбинський М.М.        | Intermezzo           | 5          |
| Нечуй-Левицький І.С.     | Кайдашева сімʼя      | 10         |
| Нечуй-Левицький І.С.     | Хмари                | 2          |
+--------------------------+----------------------+------------+

Можна використовувати інші варіанти запису запиту:

SELECT author, title, amount AS Кількість
FROM book
WHERE price < 750
ORDER BY author, Кількість DESC;

SELECT author, title, amount AS Кількість
FROM book
WHERE price < 750
ORDER BY 1, 3 DESC;

Важливо! Якщо назви стовпців укладені в лапки, при сортуванні їх використовують без лапок.

Завдання

Вивести автора і назву книг, кількість яких від 2 до 14 (включно). Інформацію відсортувати спочатку за авторами (у зворотньому алфавітному порядку), а потім за назвами книг (за зростанням).
