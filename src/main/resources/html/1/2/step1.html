Вибірка всіх даних із таблиці

Для того щоб відібрати всі дані з таблиці, використовується SQL-запит такої структури:

  • ключове слово SELECT;
  • символ * ;
  • ключове слово FROM;
  • назва таблиці.

Результатом є таблиця, в яку включено всі рядки та стовпці зазначеної у запиті таблиці.

Приклад

Вивести всі записи таблиці book.

Запит:

SELECT * FROM book;

Завдання

Виведіть інформацію про всі книги, що зберігаються на складі.

Для цього:

  • Напишіть SQL-запит у вікні коду;
  • Натисніть кнопку Відправити для перевірки;
  • Якщо запит працює неправильно — виправте його та знову відправте на перевірку.

ВАЖЛИВО! У вікні коду можна використовувати коментарі для збереження різних варіантів запитів або пояснень. Коментарі беруться у /* і */
