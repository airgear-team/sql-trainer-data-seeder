Зміст уроку

У цьому уроці буде розглянуто синтаксис і семантика наступних SQL-запитів:

  • вибірка всіх даних із таблиці;
  • вибірка даних із окремих стовпців;
  • присвоєння нових імен стовпцям під час формування вибірки;
  • створення обчислюваних стовпців;
  • обчислювані стовпці, математичні функції;
  • обчислювані стовпці, логічні функції;
  • вибірка даних за простим умовним виразом;
  • вибірка даних із використанням логічних виразів та операцій;
  • вибірка даних, оператори BETWEEN, IN;
  • вибірка даних із сортуванням;
  • вибірка текстових даних за шаблоном, оператор LIKE.
