Вибірка даних, оператор LIKE

Оператор LIKE використовується для порівняння рядків. На відміну від операторів відношення = (рівно) і <> (не рівно), LIKE дозволяє порівнювати рядки не за повним співпадінням, а за шаблоном. Шаблон може включати звичайні символи і символи-шаблони. При порівнянні з шаблоном, звичайні символи повинні точно співпадати з символами у рядку. Символи-шаблони можуть співпадати з будь-якими елементами рядка.

Символ-шаблон Опис Приклад
% Будь-який рядок з нульовою або більшою кількістю символів SELECT * FROM book WHERE author LIKE '%М.%'
знайде всі книги, у авторів яких ініціали містять «М.»
_ (підкреслення) Будь-який один символ SELECT * FROM book WHERE title LIKE 'Поэм_'
знайде книги, назва яких складається з певної кількості символів

Приклад 1

Вивести назви книг, що починаються з літери «К».

Запит:

SELECT title
FROM book
WHERE title LIKE 'К%';

Результат:

+----------------------+
| title                |
+----------------------+
| Кайдашева сімʼя      |
| Кобзар               |
| Капітанська дочка    |
+----------------------+
Пояснення

Великі і малі літери в рядках вважаються рівноцінними.

Приклад 2

Вивести назви книг, що складаються рівно з 5 символів.

Запит:

SELECT title FROM book
WHERE title LIKE "_____";

Результат:

+-------+
| title |
+-------+
| Кобзар |
| Маруся |
+-------+
Пояснення

Для позначення будь-якого одного символу використовується "_", відповідно для позначення 5 символів використовують 5 підряд символів підкреслення.

Приклад 3

Вивести книги, назва яких довша за 5 символів:

Запит:

SELECT title FROM book
WHERE title LIKE "______%";

Результат:

+------------------------+
| title                  |
+------------------------+
| Кайдашева сімʼя        |
| Тіні забутих предків   |
| Лісова пісня           |
| Капітанська дочка      |
+------------------------+
Пояснення

Щоб вивести назви, які складаються з будь-якої кількості символів після LIKE, можна використати шаблон "%", який відбирає рядки з будь-якою кількістю символів, включно з порожніми.

Приклад 4

Вивести книги, у назві яких є слово "і".

Запит:

SELECT title FROM book
WHERE   title LIKE "_% і _%"
    OR title LIKE "і _%"
    OR title LIKE "_% і"
    OR title LIKE "і";

Результат:

+------------------------+
| title                  |
+------------------------+
| Кайдашева сімʼя        |
| Тіні забутих предків   |
| Лісова пісня           |
+------------------------+
Пояснення

Слово "і" може розташовуватись у назві в будь-якій позиції. Для цього використовуються шаблони "_% і _%", "і _%", "_% і" або "і".

Приклад 5

Вивести назви книг, які складаються рівно з одного слова.

Запит:

SELECT title FROM book
WHERE title NOT LIKE "% %";

Результат:

+---------+
| title   |
+---------+
| Кобзар  |
| Маруся  |
+---------+
Пояснення

Відсутність пробілу в назві означає, що вона складається з одного слова. Для цього використовується оператор NOT LIKE.

Завдання

Вивести назву і автора тих книг, назва яких складається з двох і більше слів, а ініціали автора містять літеру «С». Інформацію відсортувати за назвою книги в алфавітному порядку.

Результат:

+------------------------+-------------------------+
| title                  | author                  |
+------------------------+-------------------------+
| Кайдашева сімʼя        | Нечуй-Левицький І.С.    |
| Тіні забутих предків   | Коцюбинський М.М.       |
| Лісова пісня           | Леся Українка           |
+------------------------+-------------------------+
Пояснення
  • При записі умови необхідно врахувати, що слово в назві обов’язково має містити хоча б один символ.
  • Ініціали в цьому завданні — це перша літера імені або по батькові, після якої стоїть крапка.
  • Літера С має бути введена в УКРАЇНСЬКІЙ розкладці.
