Вибірка даних, оператор LIKE

Оператор LIKE використовується для порівняння рядків. На відміну від операторів відношення = (рівно) і <> (не рівно), LIKE дозволяє порівнювати рядки не за повним співпадінням, а за шаблоном. Шаблон може включати звичайні символи і символи-шаблони. При порівнянні з шаблоном, звичайні символи повинні точно співпадати з символами у рядку. Символи-шаблони можуть співпадати з будь-якими елементами рядка.

Символ-шаблон Опис Приклад
% Будь-який рядок з нульовою або більшою кількістю символів SELECT * FROM book WHERE author LIKE '%М.%'
знайде всі книги, у авторів яких ініціали містять «М.»
_ (підкреслення) Будь-який один символ SELECT * FROM book WHERE title LIKE 'Поэм_'
знайде книги, назва яких складається з певної кількості символів

Приклад 1

Вивести назви книг, що починаються з літери «К».

Запит:

SELECT title
FROM book
WHERE title LIKE 'К%';

Результат:

+----------------------+
| title                |
+----------------------+
| Кайдашева сімʼя      |
| Хмари                |
+----------------------+

Приклад 2

Вивести назви книг, що складаються рівно з 5 символів.

Запит:

SELECT title FROM book
WHERE title LIKE "_____";

Результат:

+-------+
| title |
+-------+
| Хмари |
+-------+

Приклад 3

Вивести книги, назва яких довша за 5 символів:

Запит:

SELECT title FROM book
WHERE title LIKE "______%";

Результат:

+------------------------+
| title                  |
+------------------------+
| Тіні забутих предків   |
| Кайдашева сімʼя        |
| Intermezzo             |
| Земля                  |
+------------------------+

Приклад 4

Вивести книги, у назві яких є слово "і".

Запит:

SELECT title FROM book
WHERE   title LIKE "_% і _%"
    OR title LIKE "і _%"
    OR title LIKE "_% і"
    OR title LIKE "і";

Результат:

+------------------------+
| title                  |
+------------------------+
| Кайдашева сімʼя        |
| Тіні забутих предків   |
+------------------------+

Приклад 5

Вивести назви книг, які складаються рівно з одного слова.

Запит:

SELECT title FROM book
WHERE title NOT LIKE "% %";

Результат:

+---------+
| title   |
+---------+
| Хмари   |
| Земля   |
+---------+
