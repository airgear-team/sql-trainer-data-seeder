Вибірка окремих стовпців

Щоб відібрати дані з окремих стовпців таблиці, використовується SQL-запит такої структури:

  • ключове слово SELECT;
  • список стовпців таблиці через кому;
  • ключове слово FROM;
  • назва таблиці.

Результатом є таблиця, до якої включено всі дані з вказаних після SELECT стовпців вихідної таблиці.

Приклад

Вивести назви книг та їхню кількість із таблиці book.

Запит:

SELECT title, amount FROM book;

Результат:

+-----------------------+--------+
| title                 | amount |
+-----------------------+--------+
| Тіні забутих предків  | 3      |
| Intermezzo            | 5      |
| Кайдашева сім’я       | 10     |
| Хмари                 | 2      |
| Земля                 | 15     |
+-----------------------+--------+

Завдання

Виберіть авторів, назви книг і їхню ціну з таблиці book.

Результат:

+--------------------------+-----------------------+--------+
| author                   | title                 | price  |
+--------------------------+-----------------------+--------+
| Коцюбинський М.М.        | Тіні забутих предків  | 670.99 |
| Коцюбинський М.М.        | Intermezzo            | 540.50 |
| Нечуй-Левицький І.С.     | Кайдашева сім’я       | 460.00 |
| Нечуй-Левицький І.С.     | Хмари                 | 799.01 |
| Кобилянська О.Ю.         | Земля                 | 650.00 |

Affected rows: 5
