Присвоєння нових імен стовпцям у вибірці

Щоб відібрати дані з певних стовпців таблиці та одночасно задати їм нові назви у результаті, використовується SQL-запит такої структури:

  • ключове слово SELECT;
  • назва стовпця;
  • ключове слово AS;
  • нове ім’я стовпця (можна українською, але як одне слово або через підкреслення);
  • кома;
  • наступний стовпець;
  • ...;
  • ключове слово FROM;
  • назва таблиці.

У одному запиті можна використовувати як імена стовпців із таблиці, так і нові назви.

Результатом буде таблиця, яка включає всі дані з вказаних після SELECT стовпців вихідної таблиці. Кожному стовпцю у результаті запиту присвоюється нове ім’я, задане після AS, або використовується ім’я стовпця вихідної таблиці, якщо AS не вказано.

Приклад

Вивести назви книг та їх кількість із таблиці book, для стовпця title задати нове ім’я Назва.

Запит:

SELECT title AS Назва, amount FROM book;

Результат:

+-----------------------+--------+
| Назва                 | amount |
+-----------------------+--------+
| Кайдашева сімʼя       | 3      |
| Тіні забутих предків  | 5      |
| Кобзар                | 10     |
| Маруся                | 2      |
| Лісова пісня          | 7      |
+-----------------------+--------+

Завдання

Виберіть назви книг та авторів із таблиці book. Для поля title задай ім’я (псевдонім) Назва, для поля author — Автор.

Результат:

| Назва                 | Автор                    |
+-----------------------+--------------------------+
| Кайдашева сімʼя       | Нечуй-Левицький І.С.     |
| Тіні забутих предків  | Коцюбинський М.М.        |
| Кобзар                | Шевченко Т.Г.            |
| Маруся                | Квітка-Основʼяненко Г.Ф. |
| Лісова пісня          | Леся Українка            |
