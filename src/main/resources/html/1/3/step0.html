Зміст уроку

SQL-запити дозволяють виконувати обчислення не лише для кожного рядка таблиці, а й для групи елементів, що розташовані в одному стовпці. Для цього використовуються групові (агрегатні) функції.

У цьому уроці розглянемо запити, які реалізують:

  • вибірку унікальних елементів стовпця;
  • вибірку даних (групові функції SUM і COUNT);
  • вибірку даних (групові функції MIN, MAX і AVG);
  • вибірку даних з обчисленнями (групові функції);
  • обчислення по таблиці загалом;
  • вибірку даних за умовою (групові функції);
  • вибірку даних за умовою (групові функції WHERE і HAVING).
