Вибірка даних, групові функції MIN, MAX та AVG

До групових функцій SQL відносяться: MIN(), MAX() та AVG(), які обчислюють мінімальне, максимальне та середнє значення елементів стовпця, що належать до групи.

Приклад

Вивести мінімальну ціну книги кожного автора

Запит:

SELECT author, MIN(price) AS min_price
FROM book
GROUP BY author;

Результат:

+-----------------------------+-----------+
| author                      | min_price |
+-----------------------------+-----------+
| Нечуй-Левицький І.С.        | 320.50    |
| Коцюбинський М.М.           | 410.00    |
| Шевченко Т.Г.               | 520.00    |
| Квітка-Основʼяненко Г.Ф.    | 300.00    |
| Леся Українка               | 450.00    |
+-----------------------------+-----------+

Завдання

Вивести прізвище та ініціали автора, мінімальну, максимальну та середню ціну книг кожного автора. Обчислювані стовпці назвати Мінімальна_ціна, Максимальна_ціна та Середня_ціна відповідно.

Результат

| author                      | Мінімальна_ціна  | Максимальна_ціна  | Середня_ціна |
+-----------------------------+------------------+-------------------+--------------+
| Нечуй-Левицький І.С.        | 320.50           | 320.50            | 320.500000   |
| Коцюбинський М.М.           | 410.00           | 410.00            | 410.000000   |
| Шевченко Т.Г.               | 520.00           | 520.00            | 520.000000   |
| Квітка-Основʼяненко Г.Ф.    | 300.00           | 300.00            | 300.000000   |
| Леся Українка               | 450.00           | 450.00            | 450.000000   |
+-----------------------------+------------------+-------------------+--------------+
