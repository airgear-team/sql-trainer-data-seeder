Вибір унікальних значень зі стовпця

Щоб отримати унікальні значення певного стовпця, використовується ключове слово DISTINCT, яке розміщується відразу після SELECT.

Приклад

Вибрати різних авторів, книги яких зберігаються в таблиці book.

Запит:

SELECT DISTINCT author
FROM book;

Результат:

Affected rows: 3
+----------------------+
| author               |
+----------------------+
| Коцюбинський М.М.    |
| Нечуй-Левицький І.С. |
| Кобилянська О.Ю.     |
+----------------------+

Інший спосіб — використання оператора GROUP BY, який групує записи з однаковими значеннями у вказаному стовпці. Стовпець, за яким проводиться групування, вказується після GROUP BY.

Завдяки GROUP BY можна також отримати унікальні значення — результат буде аналогічний як при використанні DISTINCT.

Запит:

SELECT author
FROM book
GROUP BY author;
