Вступ до курсу по SQL

SQL (Structured Query Language – мова структурованих запитів) — це стандартна мова роботи з реляційними базами даних. Вона використовується для створення, зміни та управління даними, що зберігаються у таблицях.

Основні поняття:

  • База даних (Database) – впорядкована сукупність даних, організованих у таблицях.
  • Таблиця (Table) – основна структура зберігання даних, складається з рядків і стовпців.
  • Рядок (Row, Record) – окремий запис у таблиці, що містить дані.
  • Стовпець (Column, Field) – атрибут таблиці, який визначає тип даних (наприклад, число, текст, дата).
  • Первинний ключ (Primary Key) – унікальний ідентифікатор рядка в таблиці.
  • Зовнішній ключ (Foreign Key) – поле, яке пов’язує одну таблицю з іншою.
  • Запит (Query) – інструкція, за допомогою якої ми отримуємо, змінюємо чи видаляємо дані.

Основні можливості SQL:

  • Створення структури даних – створення таблиць, визначення типів даних, зв’язків між таблицями (CREATE, ALTER, DROP).
  • Маніпуляція даними – додавання, оновлення, видалення даних (INSERT, UPDATE, DELETE).
  • Отримання даних – пошук і вибірка даних за умовами (SELECT, WHERE, ORDER BY, GROUP BY).
  • Керування доступом – контроль прав користувачів (GRANT, REVOKE).

SQL є універсальною мовою, яку підтримують практично всі системи управління базами даних: MySQL, PostgreSQL, Oracle, Microsoft SQL Server, SQLite та інші.

У цьому курсі ви навчитеся працювати з таблицями, будувати запити, аналізувати дані та створювати власні бази даних.
